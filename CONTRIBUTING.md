# 📋 Guia de Contribuição

Obrigado por estar interessado em contribuir ao **IRS Automator**! 🎉

## 🚀 Como Começar

### 1. Fork e Clone
```bash
git clone https://github.com/SEU_USERNAME/IRS.git
cd IRS
```

### 2. Criar Branch
```bash
git checkout -b feature/sua-feature-aqui
# ou para bugfix:
git checkout -b bugfix/descricao-bug
```

### 3. Fazer Mudanças
- Mantenha consistência no código
- Teste localmente as correções
- Adicione comentários explicativos onde necessário

### 4. Commit e Push
```bash
git add .
git commit -m "Descrição clara do que mudou"
git push origin feature/sua-feature-aqui
```

### 5. Pull Request
- Abra um PR com descrição detalhada
- Referencie qualquer issue relacionada
- Aguarde revisão

---

## 📐 Padrões de Código

- **Python**: Seguir PEP 8
- **Nomes de variáveis**: `variavel_descritiva` (snake_case)
- **Comentários**: Português claro e conciso
- **Commits**: Mensagens em Português, descritivas

---

## 🧪 Antes de Submeter

- [ ] Testou localmente a funcionalidade?
- [ ] Verificou se não quebra funcionalidade existente?
- [ ] Removeu dados confidenciais/credenciais?
- [ ] Atualizou documentação se necessário?

---

## 🚫 NÃO Contribuir Com

- ❌ Dados pessoais (NIFs, IBANs, emails reais)
- ❌ Credenciais ou tokens
- ❌ Caminhos de sistemas absolutos
- ❌ Código executável de origem desconhecida

---

- Veja exemplos no código existente
